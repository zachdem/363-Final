README - How to setup and run java application:

1. Open the FinalProject363 folder in Eclipse as a project file->Open Project From File System

2. Import the mysql-connector-java-8.0.21.jar file into the project as demonstrated in class

3. Open cs363Final.java and hit the green play button to run the application

4. A prompt should appear asking for a username and password, username = "coms363" password = "coms363"

5. After logging in please follow the prompts on the screen to run all the queries for project
